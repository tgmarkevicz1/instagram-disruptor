# Instagram Disruptor (Full Project)

## Structure

- /extension → Chrome extension
- server.js → Node helper app

## Setup

### 1. Run helper
node server.js

### 2. Load extension
Go to chrome://extensions → Load unpacked → select /extension

## Features
- Random 3–5 min trigger
- Plays video
- Tracks time spent
- Unmutes system via helper

## Notes
Mac uses osascript
Windows requires nircmd
