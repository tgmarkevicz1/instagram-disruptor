# Instagram Disruptor

A Chrome extension + local Node.js helper that tracks your Instagram time and plays a break reminder after 3–5 minutes.

---

## Quick Start (automated)

```bash
# 1. Run the setup script once
bash setup.sh

# 2. Start the helper server
node server.js   # or ./start_helper.sh

# 3. Load the extension in Chrome
#    chrome://extensions → Developer mode → Load unpacked → select /extension
```

---

## Project Structure

```
instagram_disruptor/
├── server.js             ← Local helper (handles system unmute)
├── setup.sh              ← One-time automated setup
├── start_helper.sh       ← Generated by setup.sh (macOS/Linux)
├── start_helper.bat      ← Generated by setup.sh (Windows)
├── package.json          ← Generated by setup.sh
└── extension/
    ├── manifest.json
    ├── background.js
    ├── content.js
    ├── popup.html
    └── popup.js
```

## Features

| Feature | Detail |
|---|---|
| Random trigger | 3–5 minutes after opening Instagram |
| Per-tab timers | Multiple Instagram tabs handled independently |
| Time tracking | Cumulative time displayed in the popup |
| System unmute | Via local helper (osascript / amixer / nircmd) |
| Video overlay | Dismissible break reminder video |
| Popup | Shows total time + helper status + Reset button |

---

## Platform Notes

| OS | Unmute method | Requirement |
|---|---|---|
| macOS | `osascript` | Built-in, nothing to install |
| Linux | `amixer` | `alsa-utils` (setup.sh installs it) |
| Windows | `nircmd.exe` | Download from nirsoft.net, place next to server.js |

---

## Permissions

| Permission | Why |
|---|---|
| `tabs` | Detect active Instagram tab, unmute it |
| `scripting` | Inject the video overlay |
| `storage` | Persist time tracking across page loads |
| `host_permissions: instagram.com` | Run content script on Instagram |
| `host_permissions: 127.0.0.1:3000` | Call the local helper server |
